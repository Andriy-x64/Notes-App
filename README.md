# Notes App

## Опис

**Notes App** — мобільний застосунок для створення та організації нотаток із вкладеною структурою папок, локальними нагадуваннями, пошуком і rich-text редагуванням.

Проєкт побудований на **Expo Router** та **TypeScript** і використовує **SQLite** для локального зберігання даних, **expo-notifications** для нагадувань, а також окремі модулі для пошуку, сортування, папок і нотаток.

## Основні можливості

- створення, редагування та видалення нотаток;
- rich-text редактор для форматування вмісту нотаток;
- створення папок і вкладених папок;
- переміщення нотаток між папками;
- закріплення нотаток;
- локальні нагадування для нотаток;
- список нагадувань із фільтрацією за статусом;
- глобальний пошук по нотатках і папках;
- сортування нотаток і папок;
- видалення кількох елементів одночасно;
- локальне збереження даних у SQLite;
- відкриття нотатки з натискання на системне сповіщення.

## Вимоги для запуску

### Обов’язково

- **Node.js**
- **npm** (або сумісний менеджер пакетів)
- середовище для запуску Expo-проєкту

### Для повноцінного запуску на Android

- **JDK 17**

- емулятор Android або фізичний пристрій

### Для запуску на iOS

- **macOS**
- **Xcode**

### Важлива примітка

Для цього проєкту **рекомендовано використовувати development build**, а не лише Expo Go.

Expo Go підходить для швидкого тестування базових сценаріїв, але він обмежений для бібліотек, які потребують власного native code або кастомної нативної конфігурації. Для повноцінної перевірки цього застосунку development build є кращим варіантом.

Примітка: у шляху до коду програми має використовуватися тільки Латиниця, інакше не буде бачити шлях що б використовувати development build.

## Залежності

Усі бібліотеки з `package.json` встановлюються через `npm install`. Окремо вручну ставити npm-пакети не потрібно.

## Встановлення

```bash
git clone <REPO_URL>
cd <PROJECT_FOLDER>
npm install
```

Якщо репозиторій уже завантажено локально, достатньо перейти в кореневу папку проєкту та виконати:

```bash
npm install
```

## Запуск

### Стандартний запуск Expo

```bash
npx expo start
```

### Запуск Android development build

```bash
npx expo run:android
```

### Запуск iOS development build

```bash
npx expo run:ios
```

Під час роботи `expo start` можна:

- натиснути `w` для запуску web;
- натиснути `a` для Android;
- натиснути `i` для iOS.

## Приклади використання

### Створити нотатку

1. Відкрийте вкладку `Нотатки`.
2. Натисніть кнопку `+`.
3. Введіть заголовок і вміст.
4. За потреби відформатуйте текст у редакторі.
5. Застосунок збереже чернетку автоматично.

### Створити нагадування з нотатки

1. Відкрийте нотатку.
2. Натисніть іконку дзвоника.
3. Оберіть дату й час.
4. Застосунок створить локальне системне сповіщення.

### Створення нагадування

1. Відкрийте вкладку **Нагадування** або екран створення нагадування.
2. Вкажіть назву, опис і дату/час.
3. Збережіть нагадування.
4. Після спрацювання сповіщення можна перейти до пов’язаної нотатки.

### Організація нотаток у папках

1. Відкрийте вкладку **Папки**.
2. Створіть нову папку.
3. За потреби створіть вкладену папку.
4. Перемістіть нотатку до потрібної папки, або створіть у обраній папці.

### Знайти нотатку або папку

1. Відкрийте вкладку `Пошук`.
2. Введіть текст для пошуку.
3. Перегляньте результати по нотатках і папках.

Примітка: якщо використовувати доступне поле пошуку у відкритій папці, то пошук відбуватимнться тільки по цій папці, та інших її вкладених елементах.

## Структура проєкту

```text
src/
  app/             # Роути Expo Router та екрани навігації
  core/            # DI, контракти, типи, база даних, налаштування
  features/        # Логіка функціональних модулів: notes, folders, reminders, search,sorting
  infrastructure/  # Реалізації для SQLite, сповіщень і пошуку
  shared/          # Спільні UI-компоненти, меню, діалоги та утиліти
  assets/          # Зображення та іконки
```
